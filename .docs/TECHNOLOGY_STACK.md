# Technology Stack

## Backend (FastAPI)
- **Framework**: FastAPI (Python 3.10+), asynchronous request handling.
- **Database ORM**: SQLAlchemy 2.x with async support.
- **Database**: SQLite for local development, PostgreSQL in production (configured via `app/core/config.py`).
- **Spaced‑Repetition Engine**: **FSRS v6** (implemented in `app/modules/deck/services/fsrs_engine.py` – uses stability, difficulty, rating to compute next due date).
- **AI Integration**: Gemini LLM via background tasks (`app/modules/ai/services/gemini_service.py`).
- **Authentication**: Cookie‑based session managed by FastAPI authentication routers.
- **Background Tasks**: FastAPI `BackgroundTasks` for AI explanation generation and audio TTS.
- **Testing**: Pytest with async fixtures.

## Frontend (React 19 + Vite)
- **Core**: React 19 with TypeScript for type safety.
- **Bundler**: Vite (fast HMR, ES‑module dev server).
- **Styling**: Tailwind v4 (utility‑first CSS) + custom CSS variables for dark‑mode.
- **State Management**: Zustand (lightweight global store) for auth, user profile, gamification counters.
- **Data Fetching**: TanStack Query v5 (caching, retries, optimistic updates) with Axios (`withCredentials: true`).
- **Routing**: React Router v6 with protected layout components.
- **Audio**: HTML5 `<audio>` element playing TTS files generated by `/generate-audio/{card_id}`.
- **UI Components**: Reusable components (`FlashcardCard`, `FlashcardMap`, `Navbar`, `ProgressBar`).
- **Accessibility**: `aria-label`s, focus‑visible utilities, icon‑based status indicators.

## DevOps / Build
- **Containerisation**: Dockerfile for backend, separate Dockerfile for frontend (multi‑stage build).
- **CI/CD**: GitHub Actions runs lint (`eslint`, `flake8`), format (`prettier`, `black`), tests, and builds Docker images.
- **Deployment**: FastAPI served by Uvicorn/Gunicorn behind Nginx; Vite build output served as static files from `app/static/dist`.

## Additional Libraries
- **FSRS**: `fsrs` Python package (v6 implementation).
- **Gemini SDK**: Custom wrapper around Google Gemini API.
- **SQLAlchemy‑utils**: JSON column handling for flexible `others` field.
- **Pydantic**: Request/response schemas and validation.
- **Python‑dotenv**: Environment variable management.
- **Tailwind Plugins**: `@tailwindcss/forms`, `@tailwindcss/typography`.

This file provides a quick reference for developers onboarding the Vocaburn project and clarifies the core technologies powering the flashcard‑only platform.
